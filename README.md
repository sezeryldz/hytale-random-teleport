# Random Teleport

A simple random teleportation plugin for Hytale servers.

## Features

- `/rtp` - Teleport to a random location 5000-9000 blocks from spawn
- **1 hour cooldown** to prevent spam
- **Aliases:** `/randomtp`, `/randomteleport`

## Installation

1. Download the latest JAR from [Releases](https://github.com/Vorlas/hytale-random-teleport/releases)
2. Place it in your server's `mods` folder
3. Restart the server
4. Use `/rtp` in-game!

## Building from Source

```bash
./gradlew shadowJar
```

The built JAR will be in `build/libs/`

## Requirements

- Hytale Server (Early Access)
- Java 25+
